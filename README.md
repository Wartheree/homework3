# **Отчет к домашнему заданию №3**
## **1. Условие задачи**
В рамках выполнения данного домашнего задания требуется разработать программу строкового калькулятора. Причём по запросу с клавиатуры в консольном приложении вводится строка, которая может содержать:
+ знаки операций +,-,*,/;
+ константы (целые или вещественные);
+ скобки ();
+ функции sin, cos, tg, ctg, exp;
+ переменную x.  

Если переменная x есть в формуле, то запрашивается её значение, после чего вычисляется результат и выводится на экран.
## **2. Схема алгоритма**
Программа строкового калькулятора работает по следующему алгоритму:
1) запись исходной формулы в постфиксной форме;
2) определение наличия переменной x в данной формуле и запрашивание её значения;
3) вычисление значения данного выражения при помощи его записи в постфиксной форме;
4) вывод значения на экран пользователя.
## **3. Контрольный расчёт**
На вход программе подается следующее выражение: cos(x-5)\*10/(2\*x-5)  
Программа запрашивает значение переменной x.  
Введём значение 5.  
Программа выводит результат "2", что является верным ответом.  
```
enter expression to calculate: cos(x-5)*10/(2*x-5)  
enter value of x: 5  
result: 2  
```
